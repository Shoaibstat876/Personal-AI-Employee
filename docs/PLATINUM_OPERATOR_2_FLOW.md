# Platinum Operator 2 Flow

1. Cloud-prepared draft is placed in Updates/Incoming
2. Local claims by move into Updates/Claimed
3. Local reviews with APPROVE, REJECT, or IGNORE
4. APPROVE creates a Gold-compatible file in Needs_Action
5. Reviewed update is archived into Updates/Processed
6. Platinum layer never executes external actions
7. Gold remains system of record
