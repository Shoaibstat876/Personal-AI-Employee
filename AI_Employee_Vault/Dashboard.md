## Dashboard Contract (v1)

### Purpose
Defines what the dashboard displays.
Definition only.
No automation implied.

### Tracked Folder Counts
- Inbox/
- Needs_Action/
- In_Progress/
- Pending_Approval/
- Done/
- Blockers/ (if present)

Counts are manually updated in Bronze.

### Last Updated Format

Last_Updated: YYYY-MM-DD HH:MM (UTC)

No automatic scanning.
No scripts.
Manual update only.
